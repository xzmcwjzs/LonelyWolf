04月25日-世界防治疟疾日(World Malaria Day)

     世界防治疟疾日的由来 世界防治疟疾日的历年主题

   最新动态
　　过去十年，全世界在与疟疾作斗争方面已经取得重大进展。自2000年以来，疟疾死亡率下降幅度超过25%，99个现有疟疾传播的国家中有50个应该可以实现世界卫生大会确定的到2015年将发病率降低75%以上的目标。扩大病媒控制措施并增加获得诊断检验包和质量有保证的药物对取得这一进展十分关键。
　　但我们尚未实现目标。疟疾仍然在全球杀死约66万人，其中主要是撒哈拉以南非洲的五岁以下儿童。每年会发生2百多万例疟疾，其中大部分从未得到检验或登记。近来，国际资金不再增加，使取得进步的速度慢了下来。而且，正在出现的药物和杀虫剂耐药很可能会逆转近年来的好形势。
　　2013年和今后数年的主题是：投资未来，击败疟疾。
　　目标：使抗疟承诺重添活力
　　世卫组织会员国在2007年世界卫生大会期间设立世界防治疟疾日。在这一天，我们要强调有必要在预防和控制疟疾方面继续投资并保持政治承诺。这一天也是一个契机：
　　受影响地区各国可以相互交流经验并相互支持；
　　新捐助者可以加入遏制疟疾全球伙伴关系；
　　研究和学术机构可以向专家和公众介绍科学进展；
　　国际伙伴、公司和基金会可以展示其努力，并思考如何进一步扩大干预措施。　
　
　　世卫组织将利用这一计划阐明疟疾构成重大卫生挑战环境下的最佳实践，并促进各国交流调整并加强控制疟疾努力的经验。

   世界防治疟疾日的由来

　　世界疟疾日由世界卫生大会在2007年5月第六十届会议上设立，旨在推动全球进行疟疾防治。2008年4月25日为首个世界疟疾日。据世界卫生组织报告，全球大约40%的人口受虐疾威胁，每年有3.5亿―5亿人感染疟疾，110万人因疟疾死亡。每天有3000儿童因患疟疾而失去生命。发病主要集中在经济相对落后、交通不便的老、少、边、穷地区。对全世界大约二分之一的人而言，疟疾迄今仍是公众健康所面临的最严重威胁之一。
